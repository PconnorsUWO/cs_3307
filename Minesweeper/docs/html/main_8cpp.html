Returns
EXIT_SUCCESS on success, EXIT_FAILURE if a critical resource is missing.

The main function creates a QApplication, checks for the existence of a critical resource (flag icon), and then instantiates and displays the main window. If the resource is missing, an error is shown and the application exits.
