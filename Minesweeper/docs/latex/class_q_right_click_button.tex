\doxysubsection{Detailed Description}
The \doxylink{class_q_right_click_button}{QRight\+Click\+Button} class. 

A subclass of QPush\+Button that emits a custom signal when the right mouse button is clicked. This allows for distinct handling of right-\/click events in the Minesweeper game.
