\doxysubsection{Detailed Description}
The \doxylink{class_main_window}{Main\+Window} class. 

Represents the main window of the Minesweeper application. It sets up the UI layout, integrates the custom title bar, manages the game board and its display, and handles user interactions such as left and right clicks on the cells.
